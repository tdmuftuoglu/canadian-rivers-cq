# canadian-rivers-cq

**Flow-regime and grain-size controls on suspended-sediment concentration–discharge behaviour and seasonal hysteresis across cold-region Canadian rivers**

## Description
This repository contains the datasets and analytical code for my research investigating the dynamics of suspended sediment in cold-climate watersheds. Specifically, I focus on how varying flow regimes and sediment grain sizes control concentration-discharge (C-Q) relationships and seasonal hysteresis patterns across Canadian rivers.

## Repository Structure
* `data/`: Contains the datasets used for the analysis.
* `notebooks/`: Jupyter notebooks (`.ipynb`) containing the complete Python code for data processing, statistical analysis, and visualization.
* `results/`: Generated figures, plots, and statistical output tables.

## Methods and Tools
I conducted all data processing, statistical analyses, and modeling for this study using Python via Google Colab. 

**Prerequisites & Libraries:**
* Python 3.x
* `pandas`
* `numpy`
* `matplotlib`
* `seaborn`
* `scipy`
