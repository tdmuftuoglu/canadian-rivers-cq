# canadian-rivers-cq

**Flow-regime and grain-size controls on suspended-sediment concentration–discharge behaviour and seasonal hysteresis across cold-region Canadian rivers**

## Description
This repository contains the analytical code for my research investigating the dynamics of suspended sediment in cold-climate watersheds. Specifically, I focus on how varying flow regimes and sediment grain sizes control concentration-discharge (C-Q) relationships and seasonal hysteresis patterns across Canadian rivers.

## Repository Structure
* `notebooks/`: A Jupyter notebook (`.ipynb`) containing the complete Python code for data fetching, processing, statistical analysis, and visualization.

## Methods and Tools
I conducted all data processing, statistical analyses, and modeling for this study using Python via Google Colab. The provided notebook is fully self-contained and automatically downloads the required raw datasets upon execution.

**Prerequisites & Libraries:**
* Python 3.x
* `pandas`
* `numpy`
* `matplotlib`
* `seaborn`
* `scipy`

## Usage
To review or reproduce my analysis:
1. Clone this repository: `git clone https://github.com/tdmuftuoglu/canadian-rivers-cq.git`
2. Open the notebook in Google Colab or your local Jupyter environment.
3. Ensure the required libraries are installed.
4. Run the notebook sequentially. (The datasets will be downloaded automatically by the script)
